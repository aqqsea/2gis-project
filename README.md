# 2GIS Reviews Analysis

## Project Overview

This project analyzes customer reviews collected from 2GIS.

## Objectives

* Collect reviews from 2GIS
* Clean and preprocess text data
* Analyze customer sentiment and feedback
* Identify common issues and strengths mentioned by customers
* Visualize key insights

## Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Jupyter Notebook

## Dataset

The dataset contains customer reviews collected from 2GIS.
